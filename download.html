Download System Rules

The System Rules library is available from GitHub and the Maven repository.

Download from GitHub

  • system-rules-1.0.1.jar
  • system-rules-1.0.1-javadoc.jar

Use in Maven

<dependency>
    <groupId>com.github.stefanbirkner</groupId>
    <artifactId>system-rules</artifactId>
    <version>1.0.1</version>
</dependency>

Use in Buildr

'com.github.stefanbirkner:system-rules:jar:1.0.1'

Use in Ivy

<dependency org="com.github.stefanbirkner" name="system-rules" rev="1.0.1" >
    <artifact name="system-rules" type="jar"/>
</dependency>

Use in Groovy Grape

@Grapes(
@Grab(group='com.github.stefanbirkner', module='system-rules', version='1.0.1')
)

Use in Grails

compile 'com.github.stefanbirkner:system-rules:1.0.1'
