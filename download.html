Download System Rules

The System Rules library is available from GitHub and the Maven repository.

Download from GitHub

  • system-rules-1.1.0.jar
  • system-rules-1.1.0-javadoc.jar
  • system-rules-1.1.0-sources.jar

Use in Maven

<dependency>
    <groupId>com.github.stefanbirkner</groupId>
    <artifactId>system-rules</artifactId>
    <version>1.1.0</version>
</dependency>

Use in Buildr

'com.github.stefanbirkner:system-rules:jar:1.1.0'

Use in Ivy

<dependency org="com.github.stefanbirkner" name="system-rules" rev="1.1.0" >
    <artifact name="system-rules" type="jar"/>
</dependency>

Use in Groovy Grape

@Grapes(
	@Grab(group='com.github.stefanbirkner', module='system-rules', version='1.1.0')
)

Use in Grails

compile 'com.github.stefanbirkner:system-rules:1.1.0'
